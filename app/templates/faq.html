Frequently Asked Questions

Quick answers to common questions.

What is Token Arena?

A fast, transparent dashboard for token markets: leaderboards, holders, charts, and competitions.

How do I try it without connecting anything?

Enable Demo Mode from the header. You can tweak the dataset size, volatility, and more.

Where does the data come from?

See Data Sources. In Demo Mode, synthetic data is generated locally.

Can I export data?

Yes, CSV export is built into the tokens table. More export options are planned for Pro.
